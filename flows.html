Shared · NexusShared.Phase2.Run(driver, mods, persist)

Per mod: skip if Url is blank or status is done; otherwise:

Opennew tab → …/mods/<id>?tab=files → status opened
ManualDom.DeepClick("button","Manual") — pierces the <mod-download-modal> shadow DOM
Manual downloadDeepClick("a[href*='/api/files/']")
Slow downloadDeepClick("button","Slow download")
Settle & closewait ~6s → done → close tab, back to collection

A failure at any step is recorded (manual-not-found / manualdl-not-found / slow-not-found / error: …) and the run moves on. persist writes the whole CSV after every status change, so the file is always a live snapshot.

Status lifecycle (per mod)

pending→ link-ok→ opened→ manual-clicked→ manualdl-clicked→ slow-clicked→ done
off-ramps (blank Url → skipped by Phase 2): link-failed duplicate · Phase 2 failures: manual-not-found manualdl-not-found slow-not-found error: …

CSV columns: Index, Name, Url, Status, UpdatedAt — UTF-8 with BOM, rewritten after every change.
